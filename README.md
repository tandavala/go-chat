# go chat

This repository contains the code for websocket chat

# the stack

- golang
- javascript
- html
- css

# how to run

1. Clone the code
```
    https://github.com/tandavala/go-chat.git
```
2. Install [gorilla websocket](https://github.com/gorilla/websocket)

```
go get github.com/gorilla/websocket
```

3. Build
```
go build -o chat
```
4. Run
```
 ./chat
```
